git-push(1)
===========

NAME
----
git-push - Update remote refs along with associated objects.


SYNOPSIS
--------
'git-push' [--all] [--force] <repository> <refspec>...

DESCRIPTION
-----------

Updates remote refs using local refs, while sending objects
necessary to complete the given refs.
