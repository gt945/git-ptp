git-add(1)
==========

NAME
----
git-add - Add files to the cache.

SYNOPSIS
--------
'git-add' <file>...

DESCRIPTION
-----------
A simple wrapper to git-update-index to add files to the cache for people used
to do "cvs add".

OPTIONS
-------
<file>...::
	Files to add to the cache.
